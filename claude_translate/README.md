### ⚠️ **試験でよく間違える問題**

1. **インデックスとスライス**: 範囲の境界（含む/含まない）
2. **ミュータブル vs イミュータブル**: リストと文字列の違い
3. **スコープ**: ローカル変数とグローバル変数

5. **import文**: 異なるimport方法の違い
